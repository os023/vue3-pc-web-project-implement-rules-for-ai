# 推荐使用的 npm 库（共用）

Vue3 PC 端项目推荐统一使用以下库，避免重复造轮子、保证风格一致。

| 用途 | 推荐库 | 说明 |
|------|--------|------|
| 时间处理 | dayjs | 日期格式化、解析、计算；禁止手写日期逻辑。详见各模式 06-时间处理 |
| 数组/对象/判断 | lodash-es | 按需引入，减少打包体积。详见各模式 07-数组对象判断 |
| 状态管理 | Pinia | Vue 官方推荐。详见各模式 05-状态管理（库模式由使用方管理） |
| 路由 | Vue Router | 与 Vue3 配套 |
| HTTP | axios 或项目约定 | 需配合拦截器统一处理公共错误 |
| 图标 | @iconify/vue 或 Iconify 生态 | 与样式规范中的资源约定一致 |
| 工具/校验等 | 按需选型 | 优先 lodash-es；其余按团队统一，避免多套同类库并存 |

新增通用能力时优先在以上范围内选型；若引入新库，需在项目内文档或本约定中补充说明。
