# C-11 注释与文档（联邦库）

联邦库下**对外导出的 API、Props、Emits** 必须有中文说明；库包需 README 说明安装、使用与按需引入方式。

- **语言**：业务代码与库对外 API 使用中文注释。
- **要求**：关键逻辑、复杂分支、**库对外导出的函数/组件用途**需注释；注释中含 `开发者说：` 的为开发者特别说明，生成代码时需重点参考。
- **文档**：每个库包 README 中说明安装、基本用法、按需引入方式及与主应用的集成示例。

与 A/B 共用规则一致，见 [shared/11-注释与文档.md](../shared/11-注释与文档.md)。
