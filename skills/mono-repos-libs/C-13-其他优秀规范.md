# C-13 其他优秀规范（联邦库）

联邦库下 API 消息体、文案、文件大小与 tests/examples 等与共用约定一致；**构建与发布**在根或各包内配置，产物为 ESM（及可选 CJS），发布前跑测试与类型检查。

- **API 消息体**：若无特殊说明，接口格式为：成功 `{ code: 200, data: any, message: string }`，失败 `{ code: number, data: any, message: string }`（若库封装接口，与主应用约定一致）。
- **文案**：口语化、具体、每句不超过 20 字；避免高深名词、被动语态与长句。
- **构建与发布**：库产物为 ESM（及可选 CJS）；构建脚本在根或各包；建议用 changesets 或 lerna version 管理版本与 CHANGELOG；发布前跑测试与类型检查。
- **项目规范**：可结合 wendui-rules 与本仓库 agent.md「需要约定的事项」及社区优秀规范按需扩展。

共用规则见 [shared/13-其他优秀规范.md](../shared/13-其他优秀规范.md)。
