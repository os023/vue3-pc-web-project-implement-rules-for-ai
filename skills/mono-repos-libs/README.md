# 联邦库模式 · Vue3 PC 端工程规范（代号 C）

适用于 pnpm + lerna 的 monorepo，侧重**共享库**（组件库、工具库、类型包）的构建、发布与复用。与 AI 问对时请遵循本规范。

## 需要约定的事项

C 开头的文件放在本目录下，**联邦库规则单独描述**；与 A/B 共用的条文见 [shared](../shared/) 目录，各 C-xx 文件中可引用。各文件仅写本模式差异与示例；与 [shared](../shared/) 相同的条文以 shared 为准，文末以「其余见 shared/xx」引用。

本模式补充：`packages/` 下为各库包（如 ui、utils、types）；每包有清晰入口（main、module、types）；组件库不内置 Pinia；涉及日期用 dayjs、工具用 lodash-es；产物为 ESM（及可选 CJS）；建议用 changesets 或 lerna version 管理版本；演示/示例放在 `examples/` 或各包内。
