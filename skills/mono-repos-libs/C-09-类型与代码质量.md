# C-09 类型与代码质量（联邦库）

联邦库下**每个库包**均发布类型（.d.ts 或 package.json 的 types 字段）；构建顺序为先 types/utils，再依赖它们的 ui。

- **类型包**：若存在独立 `packages/types`，被其他库包与主应用引用；组件库、工具库自身也可包含类型定义，通过 package.json 的 types/exports 暴露。
- **版本与变更**：建议用 changesets 或 lerna version 管理。

其余见 [shared/09-类型与代码质量.md](../shared/09-类型与代码质量.md)。
