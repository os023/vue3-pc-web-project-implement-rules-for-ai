# C-07 数组、对象与判断（联邦库）

联邦库内数组、对象操作及常用判断**优先使用 lodash-es**，按需引入以控制库产物体积。

- **优先使用 lodash-es**：避免重复造轮子；从 `lodash-es` 按函数引入。
- **示例**：`import { get, debounce } from 'lodash-es'`。
- **对外 API**：工具库对外暴露的函数可内部使用 lodash-es，不直接 re-export 整库，避免使用方打包膨胀。

与 A/B 共用规则一致，见 [shared/07-数组对象判断.md](../shared/07-数组对象判断.md)。
