# C-10 样式规范（联邦库）

联邦库下**组件库**的样式与共用约定一致；类名以 `os023-` 为前缀，支持主题变量或配置，避免与使用方冲突。

- **命名**：自定义 CSS 类以 `os023-` 为前缀（与 wendui 约定一致）；组件库样式与单项目/联邦项目约定一致。
- **作用域**：单文件组件使用 `scoped`；组件库若带全局样式，需可被主题覆盖或通过变量配置。
- **禁止与遵守**：配色、布局、图片资源等与共用规则一致；库文档中说明主题变量与覆盖方式。

与 A/B 共用规则一致，见 [shared/10-样式规范.md](../shared/10-样式规范.md)。
