# C-03 公共组件规范（联邦库）

联邦库下「公共组件」指**组件库包**（如 `packages/ui`）内对外暴露的 Vue 组件；无业务 API/store，通过 props/events 与使用方通信。

- **位置**：组件库包内 `src/components/`，按组件名建子目录（如 `Button/`），含组件文件与样式。
- **导出**：通过 `src/index.ts` 统一导出；支持按需引入（如 `import { Button } from '@repo/ui'`）。
- **样式**：类名使用项目约定的前缀；支持主题/变量覆盖，避免与使用方样式冲突；库包 README 说明安装、使用与按需引入方式。
- 命名、Props/Emits、边界、文档见 [shared/03-公共组件规范.md](../shared/03-公共组件规范.md)。

其余见 [shared/03-公共组件规范.md](../shared/03-公共组件规范.md)。
