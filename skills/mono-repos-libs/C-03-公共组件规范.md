# C-03 公共组件规范（联邦库）

联邦库下「公共组件」指**组件库包**（如 `packages/ui`）内对外暴露的 Vue 组件；无业务 API/store，通过 props/events 与使用方通信。

- **位置**：组件库包内 `src/components/`，按组件名建子目录（如 `Button/`），含组件文件与样式。
- **命名**：PascalCase，多词（如 `UserAvatar`、`DataTable`）；导出名与目录名一致。
- **Props/Emits**：完整类型定义；提供默认值与文档；不依赖业务 API/store；可依赖本 monorepo 内 types、utils 包。
- **边界**：组件库不内置 Pinia、不直接请求业务接口；敏感配置（如 baseURL）由使用方注入。
- **导出**：通过 `src/index.ts` 统一导出；支持按需引入（如 `import { Button } from '@repo/ui'`）。
- **文档**：组件顶部用中文注释说明用途、主要 props；库包 README 说明安装、使用与按需引入方式。
- **样式**：类名以 `os023-` 为前缀；支持主题/变量覆盖，避免与使用方样式冲突。

共用规则（命名、Props、边界、文档）见 [shared/03-公共组件规范.md](../shared/03-公共组件规范.md)。
