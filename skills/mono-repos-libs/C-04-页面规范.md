# C-04 页面规范（联邦库）

联邦库模式以**库**为主，库包本身**无「页面」**；若有演示或文档站点，作为独立应用或 `examples/` 存在，用于开发与文档。

- **库内**：不定义路由、不提供「页面」组件；仅提供可被主应用复用的组件、工具、类型。主应用通过安装库包、在自身路由中挂载页面组件的方式使用。
- **演示/文档**：若需展示库的用法，可在根目录或独立包中建 `examples/` 或 `packages/dev-app`，内为完整 Vue 应用，路由与页面在该应用内配置。
- **与主应用集成**：库的 README 或文档需说明如何在主应用中引入、挂载与路由配置示例。

其余见 [shared/04-页面规范.md](../shared/04-页面规范.md)；「页面」职责由使用库的主应用承担。
