# C-06 时间处理（联邦库）

联邦库内涉及日期的逻辑（如组件内展示、工具函数）**统一使用 dayjs**，禁止手写日期格式化/解析。

- **统一使用 dayjs**：所有日期格式化、解析、计算均用 dayjs。
- **按需引入**：如 `import dayjs from 'dayjs'`，需插件时单独引入（如 `customParseFormat`、`utc`）。
- **依赖**：dayjs 在库包中声明为 dependency 或 peerDependency，与主应用一致即可；若封装日期工具，内部仅用 dayjs。

与 A/B 共用规则一致，见 [shared/06-时间处理.md](../shared/06-时间处理.md)。
