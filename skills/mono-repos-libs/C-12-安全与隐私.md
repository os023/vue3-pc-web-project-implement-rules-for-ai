# C-12 安全与隐私（联邦库）

联邦库内**不包含**密码、密钥、加密证书等；不硬编码环境相关敏感信息；若库依赖含敏感信息的后端，仅通过主应用注入（如 baseURL、token），库只做接口封装与类型定义。

- **禁止**：库内不包含密码、密钥、加密证书等；不硬编码环境相关敏感信息。
- **API 封装**：若库封装请求，baseURL、token 等由主应用通过 props 或配置注入；库只做请求封装与类型定义。
- **前端**：库不读写主应用本地存储中的敏感信息；必要时提供脱敏展示工具函数，由主应用调用。

与 A/B 共用规则一致，见 [shared/12-安全与隐私.md](../shared/12-安全与隐私.md)。
