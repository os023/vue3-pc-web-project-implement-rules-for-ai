# A-04 页面规范（单项目）

单项目下仅一套路由与一套页面树，全部在 `src/views/` 与 `src/router/` 内；组织、职责、路由约定见 [shared/04-页面规范.md](../shared/04-页面规范.md)。

### 示例（Vue3 页面：script setup + 异步兜底）

```vue
<!-- src/views/user/list/index.vue -->
<script setup lang="ts">
import { ref, onMounted } from 'vue'
import { getUserList } from '@/api/user'

const list = ref<{ id: number; name: string }[]>([])
const loading = ref(false)

onMounted(async () => {
  loading.value = true
  try {
    const res = await getUserList()
    list.value = res?.data ?? []
  } catch {
    list.value = [] // 明确兜底
  } finally {
    loading.value = false
  }
})
</script>

<template>
  <div class="os023-page os023-user-list">
    <div v-if="loading">加载中...</div>
    <ul v-else>
      <li v-for="item in list" :key="item.id">{{ item.name }}</li>
    </ul>
  </div>
</template>
```
