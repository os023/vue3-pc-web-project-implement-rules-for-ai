# A-04 页面规范（单项目）

单项目下仅一套路由与一套页面树，全部在 `src/views/` 与 `src/router/` 内。

- **组织**：一个页面一个目录，目录名与路由 path 对应；入口文件统一为 `index.vue`（或与路由 name 一致）。
- **示例**：`views/user/list/index.vue` 对应路由 `/user/list`。
- **职责**：页面负责数据拉取、状态与用户操作；复杂逻辑抽到 `composables` 或 `utils`。
- **路由**：集中配置在 `src/router/`，与 `views` 目录结构一一对应，便于维护；单应用无多入口路由。

### 示例（Vue3 页面：script setup + 异步兜底）

```vue
<!-- src/views/user/list/index.vue -->
<script setup lang="ts">
import { ref, onMounted } from 'vue'
import { getUserList } from '@/api/user'

const list = ref<{ id: number; name: string }[]>([])
const loading = ref(false)

onMounted(async () => {
  loading.value = true
  try {
    const res = await getUserList()
    list.value = res?.data ?? []
  } catch {
    list.value = [] // 明确兜底
  } finally {
    loading.value = false
  }
})
</script>

<template>
  <div class="os023-page os023-user-list">
    <div v-if="loading">加载中...</div>
    <ul v-else>
      <li v-for="item in list" :key="item.id">{{ item.name }}</li>
    </ul>
  </div>
</template>
```
