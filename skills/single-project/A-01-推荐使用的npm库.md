# A-01 推荐使用的 npm 库（单项目）

单项目模式下，推荐在**本仓库内**统一使用以下库，避免重复造轮子、保证风格一致。

| 用途 | 推荐库 | 说明 |
|------|--------|------|
| 时间处理 | dayjs | 日期格式化、解析、计算；禁止手写日期逻辑。详见 [A-06-时间处理.md](A-06-时间处理.md) |
| 数组/对象/判断 | lodash-es | 按需引入，减少打包体积。详见 [A-07-数组对象判断.md](A-07-数组对象判断.md) |
| 状态管理 | Pinia | Vue 官方推荐。详见 [A-05-状态管理.md](A-05-状态管理.md) |
| 路由 | Vue Router | 与 Vue3 配套，在 `src/router/` 集中配置 |
| HTTP | axios 或项目约定 | 需配合拦截器统一处理公共错误 |
| 图标 | @iconify/vue 或 Iconify 生态 | 与 [A-10-样式规范](A-10-样式规范.md) 中的资源约定一致 |
| 工具/校验等 | 按需选型 | 优先 lodash-es；其余按团队统一，避免多套同类库并存 |

- 单项目仅一个应用、一套依赖：所有上述库在根 `package.json` 中安装即可。
- 新增通用能力时优先在以上范围内选型；若引入新库，需在项目内文档或本约定中补充说明。

共用规则（与 B/C 一致的部分）见 [shared/01-推荐使用的npm库.md](../shared/01-推荐使用的npm库.md)。
