# A-10 样式规范（单项目）

单项目下全局样式、变量、主题均在 `src/styles/`，组件样式在各自单文件内。

- **命名**：自定义 CSS 类以 `os023-` 为前缀（与 wendui 约定一致）。
- **作用域**：单文件组件使用 `scoped`；全局样式、变量、主题放在 `src/styles/`。
- **禁止**：紫色/靛蓝/蓝紫渐变、纯平背景（需噪点或渐变）、三卡片布局、完美居中对齐、等宽多栏（需不对称）。不使用 Tailwind 默认色板。
- **资源**：图标用 Iconify，占位图用 Picsum Photos，真实图片用 Pexels，插画用 unDraw。

与 B/C 共用规则一致，见 [shared/10-样式规范.md](../shared/10-样式规范.md)。
