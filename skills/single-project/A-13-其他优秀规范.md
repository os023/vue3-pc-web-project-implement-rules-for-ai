# A-13 其他优秀规范（单项目）

单项目下接口格式、文案风格与项目规范与共用约定一致，可按需结合 wendui-rules 与社区规范扩展。

- **API 消息体**：若无特殊说明，接口格式为：成功 `{ code: 200, data: any, message: string }`，失败 `{ code: number, data: any, message: string }`。
- **文案**：口语化、具体、每句不超过 20 字；避免高深名词、被动语态与长句。
- **项目规范**：可结合 [wendui-rules](https://raw.githubusercontent.com/os023/wendui-rules/refs/heads/main/README.md) 与本仓库 agent.md 的「需要约定的事项」及社区优秀规范按需扩展。

与 B/C 共用规则一致，见 [shared/13-其他优秀规范.md](../shared/13-其他优秀规范.md)。
