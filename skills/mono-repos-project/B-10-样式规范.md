# B-10 样式规范（联邦项目）

联邦项目下各 app 的全局样式在各自 `src/styles/`；**shared-ui** 的样式需可被主题/变量覆盖，类名与共用约定一致。

- **命名**：自定义 CSS 类以 `os023-` 为前缀（与 wendui 约定一致）；shared-ui 中组件样式也遵循该前缀。
- **作用域**：单文件组件使用 `scoped`；各 app 的全局样式、变量、主题在各自 `src/styles/`；shared-ui 若带样式，需支持主题变量或配置。
- **禁止与遵守**：与共用规则一致（配色、布局、图片系统等）。

与 A/C 共用规则一致，见 [shared/10-样式规范.md](../shared/10-样式规范.md)。
