# B-04 页面规范（联邦项目）

联邦项目下**每个应用**独立拥有自己的页面树与路由，无跨应用共用「页面」。各 app 内与单项目一致；组织、职责、路由约定见 [shared/04-页面规范.md](../shared/04-页面规范.md)。各 app 独立配置在自身 `src/router/`，与**该 app** 的 `views/` 对应。

### 示例（Vue3 页面：script setup + try/catch 兜底）

```vue
<!-- packages/app-admin/src/views/user/list/index.vue -->
<script setup lang="ts">
import { ref, onMounted } from 'vue'
import { getUserList } from '@/api/user' // 本 app 内 api

const list = ref<{ id: number; name: string }[]>([])
const loading = ref(false)

onMounted(async () => {
  loading.value = true
  try {
    const res = await getUserList()
    list.value = res?.data ?? []
  } catch {
    list.value = []
  } finally {
    loading.value = false
  }
})
</script>

<template>
  <div class="os023-page os023-user-list">
    <div v-if="loading">加载中...</div>
    <ul v-else>
      <li v-for="item in list" :key="item.id">{{ item.name }}</li>
    </ul>
  </div>
</template>
```
