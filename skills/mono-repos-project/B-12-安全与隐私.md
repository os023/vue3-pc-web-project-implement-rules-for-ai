# B-12 安全与隐私（联邦项目）

联邦项目下**全仓库**禁止硬编码密码、密钥、加密证书；各 app 的敏感信息与接口安全需各自保证，共享包不包含环境相关密钥。

- **禁止**：将密码、密钥、加密证书等硬编码到代码中；全仓库统一要求。
- **API**：编写或对接 API 时避免明显的信息安全问题（如敏感参数暴露、越权）；各 app 的接口与前端需各自保证。
- **前端**：敏感信息不落本地存储；必要时脱敏展示；共享包不包含 token、baseURL 等环境相关敏感信息，由各 app 注入。

与 A/C 共用规则一致，见 [shared/12-安全与隐私.md](../shared/12-安全与隐私.md)。
