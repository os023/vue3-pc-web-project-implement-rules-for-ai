# B-07 数组、对象与判断（联邦项目）

联邦项目下全 monorepo **优先使用 lodash-es**；共享封装放在 shared-utils，各 app 按需引入。

- **优先使用 lodash-es**：数组、对象操作及常用判断用 lodash-es，避免重复造轮子。
- **按需引入**：从 `lodash-es` 按函数引入，减少打包体积，例如：`import { get, debounce } from 'lodash-es'`。
- **共享**：若 shared-utils 中有通用数据处理，内部使用 lodash-es，对外提供统一 API；各 app 通过 workspace 引用，避免各自重复安装。

与 A/C 共用规则一致，见 [shared/07-数组对象判断.md](../shared/07-数组对象判断.md)。
