# B-06 时间处理（联邦项目）

联邦项目下全 monorepo 内**统一使用 dayjs**，各 app 与 shared-utils 均禁止手写日期逻辑。

- **统一使用 dayjs**：所有日期格式化、解析、计算均用 dayjs，禁止手写日期格式化/解析逻辑。
- **按需引入**：如 `import dayjs from 'dayjs'`，需插件时单独引入（如 `customParseFormat`、`utc`）。
- **依赖**：可在根或各 app、shared-utils 中安装 dayjs；若 shared-utils 封装日期工具，内部仍用 dayjs。

与 A/C 共用规则一致，见 [shared/06-时间处理.md](../shared/06-时间处理.md)。
