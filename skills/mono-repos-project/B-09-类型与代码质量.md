# B-09 类型与代码质量（联邦项目）

联邦项目下**共享类型**放在 `packages/shared-types`，各 app 与 shared-ui、shared-utils 通过 workspace 引用；各 app 内仅本应用使用的类型放在该 app 的 `src/types/` 或文件内。

- **TypeScript**：全仓库开启严格模式；对外接口、API 入参/出参、store 状态均定义类型。
- **接口定义**：跨包共用的放在 `packages/shared-types`；单 app 内多文件共用的放在该 app 的 `src/types/`；单文件内使用的放在文件顶部或同文件内。
- **代码质量**：根目录可配置 ESLint + Prettier，各包继承或覆盖；遵循 Vue3 与 TypeScript 最佳实践；注意工具函数复用，避免大段重复代码。
- **依赖**：内部包引用使用 workspace 协议（如 `"@repo/shared-utils": "workspace:*"`）。

共用规则见 [shared/09-类型与代码质量.md](../shared/09-类型与代码质量.md)。
