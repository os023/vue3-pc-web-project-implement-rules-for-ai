# 联邦项目模式 · Vue3 PC 端工程规范（代号 B）

适用于 pnpm + lerna 的 monorepo，多应用/多可运行项目共享公共组件、工具与类型。与 AI 问对时请遵循本规范。

## 需要约定的事项

本模式补充：根目录 `packages/` 下 app-* 为应用、shared-* 为共享包；各 app 内 `src/` 结构同单项目；内部包使用 workspace 协议；先构建 shared-*，再构建 app-*；各 app 独立使用 Pinia，状态不跨 app 共享。
