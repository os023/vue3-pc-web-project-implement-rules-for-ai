# B-13 其他优秀规范（联邦项目）

联邦项目下 API 消息体、文案风格、文件大小与 tests/examples 等与共用约定一致；构建与脚本在根通过 lerna/pnpm 编排。

- **API 消息体**：若无特殊说明，接口格式为：成功 `{ code: 200, data: any, message: string }`，失败 `{ code: number, data: any, message: string }`。
- **文案**：口语化、具体、每句不超过 20 字；避免高深名词、被动语态与长句。
- **构建与脚本**：根 `package.json` 中通过 lerna/pnpm 编排各包构建顺序；先构建 shared-*，再构建 app-*。
- **项目规范**：可结合 wendui-rules 与本仓库 agent.md「需要约定的事项」及社区优秀规范按需扩展。

共用规则见 [shared/13-其他优秀规范.md](../shared/13-其他优秀规范.md)。
